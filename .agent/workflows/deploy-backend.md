// turbo-all
1. Ensure the user has Node.js and Python installed.
2. Open PowerShell in the root of the `ecovision` folder.
3. Run the setup script:
```powershell
.\setup_backend.ps1
```
4. Verify the backend is running:
```powershell
pm2 status
```
